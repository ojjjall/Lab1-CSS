Web Development Glossary

DOM (Document Object Model)
A programming interface for web documents. It represents the page as a tree of objects so JavaScript can manipulate structure, style, and content dynamically.
API (Application Programming Interface)
A set of rules that allows one software application to communicate with another. For example, a weather app uses a weather API to fetch forecast data.
Responsive Design
A design methodology where a web page adapts its layout and content to different screen sizes using fluid grids, flexible images, and media queries.
Semantic HTML
Writing HTML using elements that convey meaning — e.g. <article>, <nav>, <header> — rather than non-descriptive containers like <div>.
Accessibility (a11y)
The practice of designing and developing websites so they can be used by everyone, including people with visual, auditory, motor, or cognitive disabilities.
Version Control
A system that records changes to files over time. Git is the most popular version control tool, allowing developers to track history, branch, and collaborate.
