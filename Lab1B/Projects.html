HOLO SHOP — 3D Furniture E-Commerce Platform

HOLO SHOP is an e-commerce web application designed to help people purchase furniture online with a unique and immersive shopping experience. The standout feature is its 3D shopping experience — customers can view furniture items in three dimensions before making a purchase, helping them visualise how a piece will look in their home.

Project Overview

Think simply. Build clearly.* — that is the philosophy behind HOLO SHOP. The platform brings together a clean, intuitive storefront with a cutting-edge 3D viewer, making it easier than ever for shoppers to make confident buying decisions without visiting a physical store.

Key Features

  • Interactive 3D product viewer — rotate, zoom and inspect furniture from all angles
  • Category browsing — Chair, Sofa, Table, Lamp, Wardrobe and more
  • Promotions system — e.g. "Get & Save 35% OFF First Order"
  • Multi-language support (English / Malay)
  • Wishlist & shopping cart functionality
  • User account registration and login
  • Search with visual camera-scan feature

Technologies Used

  • HTML5 & CSS3 — page structure and styling
  • JavaScript — interactivity and 3D rendering
  • MySQL — product catalogue and user database
  • Java — back-end logic & server-side processing

* This is the tagline displayed in the HOLO SHOP navigation bar.
