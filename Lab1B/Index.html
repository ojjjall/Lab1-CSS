Web Development Glossary

DOM
Document Object Model — a tree-structured in-memory representation of an HTML document that JavaScript can read and manipulate.
API
Application Programming Interface — a contract that defines how two pieces of software communicate with each other.
Responsive Design
A design approach where a webpage adapts its layout and content to fit different screen sizes and orientations.
Semantic HTML
Using HTML elements that convey meaning about the content they contain, such as <article>, <nav>, and <section>, rather than generic <div> tags.
Accessibility (a11y)
The practice of building websites that can be used by people with a range of disabilities, including visual, auditory, motor, and cognitive impairments.
